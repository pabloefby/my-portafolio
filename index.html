The Otherside

Sitio web diseñado para que usuarios de diferentes partes de la República Mexicana compartan anécdotas, historias, leyendas y experiencias relacionadas con temas paranormales. Entre las categorías principales se encuentran: leyendas, avistamientos de extraterrestres, fantasmas, conspiraciones, entre otras.

Programador fullstack y bases de datos.

Sobre mi

Soy un programador Fullstack con más de 5 años de experiencia en desarrollo web, aplicaciones de escritorio y soluciones cloud. Mi carrera se ha centrado en el ecosistema JavaScript (Node.js, React) y Python, donde me apasiona crear soluciones innovadoras, robustas y eficientes que no solo cumplan con los requisitos técnicos, sino que también mejoren la vida de los usuarios mediante interfaces intuitivas y un rendimiento optimizado. Busco constantemente nuevas tecnologías y metodologías para aplicar a proyectos desafiantes.

Habilidades tecnicas

Frontend:

  • Lenguajes: HTML, CSS, JavaScript
  • Frameworks/Librerías: React, Vue.js, Angular
  • Herramientas: Webpack, Babel, npm/yarn

Backend y bases de datos:

  • Lenguajes: Node.js, Python(Django), PHP
  • Bases de Datos: PostgreSQL, MongoDB, MySQL, Firebase.
  • APIs: RESTful API Design, GraphQL

Infraestructura y DevOps:

  • Plataformas Cloud: AWS (EC2, S3, Lambda), Google Cloud Platform (GCP).
  • Contenedores: Docker, Kubernetes (Conocimiento básico).
  • Herramientas: Git, CI/CD (GitHub Actions/GitLab CI).

Virtudes y destrezas

  • Resolución de Problemas: Capacidad demostrada para analizar problemas complejos, desglosarlos en componentes manejables e implementar soluciones escalables y bien documentadas.
  • Aprendizaje Rápido: Habilidad para dominar rápidamente nuevos lenguajes, frameworks y herramientas tecnológicas, adaptándome a los cambios del sector.
  • Comunicación Técnica: Destreza para comunicar conceptos técnicos complejos a audiencias no técnicas, y para colaborar eficazmente en equipos multidisciplinarios.
  • Atención al Detalle: Foco en la calidad del código, las pruebas unitarias y de integración, y en la usabilidad de la interfaz de usuario.
